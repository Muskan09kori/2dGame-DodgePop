# 2dGame-DodgePop
Dodge Pop is a fast-paced 2D casual dodge game built with Unity. The player controls a small character in a lively cartoon-style world, dodging falling objects and obstacles for as long as possible. The goal is simple — survive and score high while the game speed and difficulty increase over time.

Features
Smooth and responsive touch-based controls
Minimalist cartoon art style
Progressive difficulty scaling
Compact and optimized for mobile devices
Built using C# and Unity 2D tools

Tech Stack
Engine: Unity (2D URP)
Language: C#
Platform: Android 
Art Style: Minimal + Cartoon 

Current Status
Core gameplay mechanics implemented
Collision, movement, and scoring system complete
UI and visual polish in progress
